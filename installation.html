Installation

The plug-in Bug-Flipper should function on any operating system which can run its required components, but has been written and tested specifically in Windows 7, 10 and Sierra. More recent versions of Python and the GIMP may be used with semi-reliability.

Windows

Download and install Python version 2.6.6 (32 bit). Restart the computer.

Download PyGtkâ€™s all-in-one installer (32 bit). Use it to install PyCairo, PyGtk, and PyGObject. Restart the computer.

Download and install the latest stable version of the GIMP (32 bit). Restart the computer.

Download BugFlipper.py or GenericBugFlipper.py from the downloads page into the folder .gimp-2.8 > plug-ins.

Also download the white-balance/color-correction plug-in based on Diego Nasetti's freeware into the plug-ins folder.

Open the GIMP (this may take a while as the GIMP loads fonts). If the installation was performed successfully, the function Bug-Flipper or Generic-Bug-Flipper will appear under the Filters tab.

Mac

Check if Python is installed by opening the Terminal and typing "whereis python." If Python is not already installed, download and install the 2.6.6 Mac Installer disk image. Restart the computer.

PyGTK?

Download and install the latest stable version of the GIMP (32 bit). Restart the computer.

Download BugFlipper.py or GenericBugFlipper.py from the downloads page. Use the Terminal to run the command "open ~/Library/Application\ Support/GIMP/2.8/plug-ins/" and move the files here.

Also download the white-balance/color-correction plug-in based on Diego Nasetti's freeware into the same folder.

Open the GIMP (this may take a while as the GIMP loads fonts). If the installation was performed successfully, the function Bug-Flipper or Generic-Bug-Flipper will appear under the Filters tab.
