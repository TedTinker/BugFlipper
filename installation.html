Installation

The plug-in Bug-Flipper should function on any operating system which can run its required components, but has been written and tested specifically on Windows 7 and 10 operating systems. Therefore, only the Windows Installation is included.

Administrative privileges are required to install software.

Windows Installation

Download and install Python version 2.6.6 (32 bit). The installer is listed as "Windows x86 MSI Installer (2.6.6)" in the python.org download list. Restart your computer after installation.

Download PyGtkâ€™s all-in-one installer (pygtk-all-in-one-2.24.0.win32-py2.6.msi). Use it to install PyCairo, PyGtk, and PyGObject. Restart the computer.

Download and install the the GIMP 2.8 (32 bit). Close the GIMP while installing plugins.

Download BugFlipper.py into the GIMP plugins folder. The location of this folder will vary depending on where you installed GIMP. On Windows 10 for GIMP 2.8 this folder is found at User//.gimp-2.8/plug-ins.

Also download the freeware White Balance/Color Correction Plug-In from Diego Nassetti into the plug-ins folder.

Open the GIMP (this may take a while as the GIMP loads fonts). If the installation was performed successfully, the function Bug-Flipper will appear under the Filters tab, as shown below.
